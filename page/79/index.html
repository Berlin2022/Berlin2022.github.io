手动安装

参考 https://easyhexo.com/3-Plugins-use-and-config/3-4-hexo-blog-encrypt/

在 Hexo 根目录的 _config.yml 启用插件

快速使用

在你需要加密的 Hexo 博文头部添加： password ， abstract ， message 。

password ：是该博客加密使用的密码。
abstract ：是该博客的摘要，会显示在博客的列表页。
message ：这个是博客查看时，密码输入框上面的描述性文字。

简化流程

使用自定义默认信息，修改Hexo 根目录的 _config.yml
