用excel函数过滤关键词，提取标题句式

背景介绍

上一篇介绍的案例中爬取到如下标题，已将所有分站标题一并采集(共计2141篇)，有一些是固定句式，有一些是编辑出来的标题，有一些是直接用长尾词做的标题，收录挺多，但是也没有多少文章，tag和列表的收录占比很高，还有一些是url?参数形式的动态抓取如http://zn.weiyuqiju.com/117.html?ivk_sa=1024320u
找到相关结果数约11,900个数字为估算值。网站管理员如需了解更准确的索引量，请使用百度站长平台

意义分析

  • 建立一个标题库，可以直接套用的标题库，那么我在需要的时候随机生成标题，可以省一部分事（起码不用弄什么教程培训标题该怎么写……拿去套用就行了）
  • 实际应用中对短尾关键字的拓展也是非常有用的。结合”伪原创工具”，以随机生成且不重复的形式，按照单个项目（比如医疗）一般1500个左右关键词，2000个库存句式，则随便可以生成百万级别的高质量伪原创内容。

初步想法

将固定句式提取出来，目前比较高效的办法是利用 find 嵌套 left/mid/right 函数，相关用法还在腾讯读书的书架里，还只是粗略浏览了一下，没有完全掌握该函数。

对应关系

分为A、B、C三组(A和B是同义关联词,C为核心关键词)，分别用“@、#、↓”进行替换
ps：已取消，因为识别困难，不方便

实际提取步骤

  1. 根据昨天晚上在微信读书上学到的知识，直接用上“=SUBSTITUTE(B63,LEFT(B63,SEARCH(“????怎么样”,B63)),”关键词A”,1)”

    • substitute+left+search函数，这里需注意我们要保留“怎么样”前面三位数，因为“质量怎么样”、“性价比怎么样”这类都是我需要提取的句式，用SEARCH函数定位的时候应该不包括其本身定位（就是减去一位）所以通配符从第四位开始“????”，这样就相当于“SEARCH()-1”。

      对于出现在尾部的这种情况需要筛选出来 结尾是“怎么样”、“怎么样？”、“怎么样?”三种情况

  2. 利用通配符“?” 可以按照字数进行过滤杂质

  3. “=VLOOKUP(“*关键字*“,B65,1,0)” 提取包含“关键字”的句式，控制通配符“*” 可以只匹配左侧或者右侧，也可以直接用excel的文本条件筛选（推荐）

  4. 按照句式进行分类，数量少的手工替换，数量多的公式匹配，Ctrl+E（智能提取）分离中英文（如果提取不精准可以用方方格子插件或者EFunction插件的ETStrExtractSub函数），然后用“substitute”把英文替换为“型号”，删除提取的中文，基本上左边部分就提取干净了，部分型号位置太靠前没有提取出来，到时候可以用（emeditor）隔行补入句式，避免单一句式太多。

  5. 提取右侧的两种方法

  6. 替换右侧句式

  7. 提取前最好将 “标签列表” 和 “产品列表” 提取出来，稍稍处理之后便能批量替换了，这里“批量替换”我们可以用dedecms的敏感词过滤功能，批量 replace 的代码还有很多高效的代码，但是我还不会写。

    另外批量替换的方式还可以采取导入Navicat_sqlite 然后在采集器里进行批量替换，也是我目前采用的方式。

定义和库

  1. 重新定义元素

    • 元素：核心词A、核心词B、关键词A、关键词B、”型号”、”翻译”、”随机优势A”、”随机优势B”、”随机作用”、”单位”、”别名”、”问题A”、”问题B”、“品牌A”、“品牌B”、“随机活动”
    • 核心词A【不包括“国家”、包括“地域”】，核心关键词B不包含
    • 关键词A(长尾关键词)、关键词B(短尾关键词)
    • 问题A(中短)、问题B(长)
    • “品牌A”（可中英文，包括国家）、“品牌B” （可中英文，不包括国家）

  2. 注意事项

    • 关键词A、关键词B 中 已将 “问题A”、”问题B” 分离
    • “问题A”、”问题B” ，在使用中已过滤 “？”，保留原句式中的问号或不带问号模式
    • 涉及“安装” 的产品词需要排除“便携式”及无需安装的设备，排除非安装型关键词
    • “哪”、“种”、“几”、“类”，“款”，“和”一般会涉及同类型不同品牌产品，需要作为“少数”，过滤出来
    • “核心词A”和“品牌”、“型号”，大概率会重复，需排除包含的句式
    • “型号”+系列，结合不符合逻辑，尽量少用
    • “别名” 指同类型产品，包括同义词
    • “官方旗舰店”、“官网”，“旗舰店” 不适合站内发表，需要作为“少数”，过滤出来
    • 句式中涉及 “哪里产的？”、“产地”、“国家” ，一律使用 “品牌B” ，“品牌A” 中可能已包含避免重复使用

句式拆解提纯

思路，以目前提取的标题句式来看，存在着部分句式过多重复使用的情况，比如【 “核心词A”质量怎么样】就被用了超过1/2的标题，那如何避免这种情况呢？

  1. 批量制作标题需要的几个库，这样才好和程序结合

    • 品牌库：变量，自定义或采集
    • 型号库：变量，自定义或采集
    • 优势库：采集，提取，有分类，可重复
    • 作用库：采集，提取，有分类，可重复
    • 问题库：采集，提取
    • 地域库：当前“省”、“市”、“区“、“地标”、“街道”
    • 前缀库：神奇的、浅析，详细解读，你必须知道的，你知道吗……
    • 精选后缀库（内容需精编）：效果图，排名，十大品牌，图片大全，使用评价，注意事项，售后服务，原理解析，规格参数，性能参数，使用视频，视频介绍
    • 通用后缀库：都在这篇文章里了！！，你必须知道的几点……
    • 通用前后缀库：解析，推荐，赏析，欣赏……
    • 否词库：注意事项，保养，维护，原理，方法，安装，使用，教程，参考资料，说明书……

  2. 固定句式库

    • 选|择|什么(??)好？：牌子，档次，品牌，风格，款式，尺寸，规格，型号，颜色……
    • (2,3)怎么样？：质量，性价比，口碑，评价，价格，费用，效果，性能，牌子
    • 关于(“关键词B”)：你知道哪些？、不不得不知道的几点、你必须知道的、你了解多少、
