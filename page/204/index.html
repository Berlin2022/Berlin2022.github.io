毛线剪藏（MaoXian Web Clipper）的安装、设置和使用

MaoXian Web Clipper可以很容易的将网上的文章（包括插图、链接、格式之类）的内容一键存成md文件，不过安装和设置稍微有点麻烦，尤其网上现有的安装教程都是老版，界面变化很大，参考价值不大，所以费了点功夫才弄好，记录一下，也为后来者方便吧

安装环境是win 10 + Chrome，当前插件的最新版是0.4.4，“本地程序”（配合maoxian使用的一个很小的本地程序，用于增强maoxian的功能）的最新版是0.2.9

安装之前一定要确定你之前有无安装过，尤其是老版本的MaoXian Web Clipper，如果之前安装过最好先卸载（按照Chrome浏览器插件的普通卸载方法即可），今天的麻烦很大程度上就是因为之前的老版本没有卸载的原因

一、安装和设置过程

整个安装过程主要参考了作者的官网和如下文章：

obsidian使用Maoxian插件完美剪藏网页文章 - 知乎前言作为有收集癖习惯的我，在网上看到不错的文章，就喜欢收藏起来，以待后面慢慢 欣赏。前几年，一直使用Evernote作为收藏软件，但是它有几个缺点让我很恼火： 其一、网页格式保存极其完好，既是优点也是缺点，在…https://zhuanlan.zhihu.com/p/400283327上文中的“配置插件”部分与新版差别较大，其他部分可以参考。

基本步骤分为4步

1、安装Chrome插件（先不要设置，更不要使用）

2、安装Ruby（因为“本地程序”是用Ruby写的）

3、安装“本地程序”并配置

4、设置插件

下面逐个说明：

1、安装Chrome插件

注意：一定用下载工具在下面的链接处下载crx文件（不要使用Chrome下载，因为Chrome会直接安装crx文件，但直接安装会导致如下的错误，而且会导致无法启用maoxian插件

https://mika-cn.github.io/maoxian-web-clipper/extension/0.4.4/maoxian-web-clipper-chromium-0.4.4.crxhttps://mika-cn.github.io/maoxian-web-clipper/extension/0.4.4/maoxian-web-clipper-chromium-0.4.4.crx

然后找到下载好的crx文件，把文件的扩展名改为“rar”（zip不行，不知道是不是我用的7zip的原因）、解压到一个文件夹（记做A），然后从Chrome的设置（chrome://settings/）中打开“扩展程序”，选择“加载已解压的扩展程序”，点选刚才解压的文件夹A，即可正常安装这个crx文件。详细步骤可以参考：

该扩展程序未列在 Chrome 网上应用店中，并可能是在您不知情的情况下添加的–解决方法_双树次仁的博客-CSDN博客_此扩展不是来自任何已知来源

2、安装Ruby

参考官网的安装方法即可：

Windows 用户

请到 这里 下载安装包，提醒一下，这里直接下载 WITHOUT DEVKIT 下面的版本即可。安装的时候确保勾选上

  • 把 ruby 添加到环境变量（该项必须勾选，否则「本地程序」无法正常启动）。
  • 使用 UTF-8 作为外部编码 （强烈建议你勾选该项，这样你在跨系统同步文件时候，才不会发生乱码问题）。（但在实际的安装中，没有发现这个选项，不知道是不是我疏忽了，请大家在安装的时候注意观察每个步骤的所有选项，必要时重新安装Ruby）

最好在安装成功后，验证一下。通过 Win+R 调出运行窗口 > 输入 cmd 回车 > 输入 ruby -v 回车 。如果输出的是 ruby 的版本信息，则说明 ruby 安装成功，并且已经添加到环境变量。

3、安装“本地程序”并配置

点击下面的链接下载“本地程序”

  • native-app-chrome.zip

火狐或 chromium浏览器请到这个页面上下载对应的压缩包：

本地程序

这个 “本地程序”类似于绿色软件，直接解压缩到要安装的路径下（我直接安装在c盘了），然后双击其中的 install.bat （注意：你点击后，会发现有个命令窗口跳出来，闪了一下就关闭了，这是正常的，程序已经安装成功了。并不是被安装了病毒）

安装完毕后配置maoxian的下载路径，也就是裁剪下来的文件存储的目录（等同于浏览器的下载目录）

二、使用方法

 在想要剪藏的页面完全加载之后，点击浏览器“扩展程序”按钮，选择maoxian web clipper，然后点击“裁剪”

然后选择要剪藏的内容，选定后按回车确认

 

 最后再指定存放位置和标签（也可以不指定标签）即可。

三、注意事项

在剪藏之前最好先浏览一遍网页，确保网页上的内容（尤其是图片）都显示出来了，也就是“完全加载了”，然后再剪藏，因为现在很多网页上的图片没被浏览到的时候都不会加载，所以，如果直接剪藏容易出现很多图片没被剪藏下来的情况

网页上的短视频没法剪藏，但会在相应位置留一个截图
