微信公众号同步助手wechatsync

一个牛B的工具，让你一次排版发布，多平台同步发布。解放自媒体人生产力，专注于写作本身！

管网：https://www.wechatsync.com/

GITHUB：wechatsync/Wechatsync

特性

  •  公众号文章同步
  •  Markdown 编辑器
  •  网页正文提取（基于 Safari 阅读模式） 可实现多平台互同步
  •  本地Markdown同步
  •  Markdown支持公众号样式定义

如何安装

1.商店安装 

商店链接打不开，试试下面第二种安装方式？

2. 压缩包安装方式

下载插件

下载安装包 打开所在的文件夹，找到下载好的WechatSync.zip文件解压

打开浏览器扩展管理页面

复制 chrome://extensions，并 粘贴到地址栏，按回车键进入扩展中心 页面。

安装插件

在扩展中心打开右上角的【开发者模式】按钮，然后刷新页面，把解压后的WechatSync文件夹拖入扩展中心

如何使用

1.多平台同步

任意网页右键菜单 点击插件提供的提取正文选项，可以做到从知乎同步到头条，头条同步到公众号，任意平台互同步

1. 任意网页右键菜单

2. 点击提取正文选项

3. 即可提取到当前页面到文章

4. 选择需要同步到渠道，即可做到互同步

2、从公众号同步

1. 网页打开任意一篇公众号文章

2. 左上角可看到同步该文章按钮

3. 点击后选择要同步到的渠道账号

4. 等待草稿同步完成，即可逐个一一发布

0:00 / 00:36

00:00

3.同步到公众号

通过任意网页提取正文，你可以从知乎或头条、36kr、虎嗅，个人博客等任意平台、同步到公众号

4.Markdown编辑器同步

从在线Markdown编辑器https://www.wechatsync.com/md/同步到知乎、头条等任意发布渠道

5.示例：

把WordPress的文章同步到公众号

下载文章同步助手 谷歌浏览器插件，登录微信公众号后即可识别到你的账号

打开一篇文章，以https://www.yixieshi.com/150917.html为例

网页右键，选中提取文章并同步

右边以删减不需要的内容，点击同步选择要同步到的公众号。

等待同步完成。

点击查看草稿

即可看到同步过去的内容，按需再调整下即可群发了
